自機を操作し、相手の弾幕攻撃をよけながら敵を倒すゲーム

---

## 基本実装項目

- **第4回で作成したmusou_kokaton.py**

## 追加実装項目

- **自機ショットの強化・操作簡易化**
  - スペースキー長押しで扇型5方向弾を連射するように変更

- **敵攻撃の多様化**
  - 敵の攻撃を4パターン作成し、確率で攻撃を選択するように変更
  - ボス用の攻撃を4パターン作成

- **自攻撃による敵攻撃の撃墜の削除**
  - 敵の攻撃を打ち落とせないように変更
